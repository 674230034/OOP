# PHP OOP - พร็อพเพอร์ตี้สแตติก (Static Properties)

**Static Properties** คือ ตัวแปรส่วนกลางของคลาสที่**ถูกแชร์ร่วมกันในทุก ๆ ออบเจกต์** ข้อมูลของตัวแปรนี้จะไม่ขึ้นอยู่กับออบเจกต์ชิ้นใดชิ้นหนึ่ง และเช่นเดียวกันกับเมธอดสแตติก คุณสามารถดึงค่าหรือแก้ไขค่าของตัวแปรนี้ได้ทันทีโดยไม่ต้องทำการสร้างวัตถุ (`new`) ก่อน

---

## 🏗️ โครงสร้างคำสั่ง (Syntax)

เราใช้คีย์เวิร์ด **`static`** นำหน้าตัวแปร และเรียกใช้ผ่านเครื่องหมาย **`::`** โดยที่**ยังคงต้องมีเครื่องหมาย `$` อยู่หน้าชื่อตัวแปรเสมอ**

```php
class ClassName {
  public static $staticValue = "ข้อมูล";
}

// วิธีเรียกใช้งานจากภายนอกคลาส
echo ClassName::$staticValue;
