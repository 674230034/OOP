# PHP OOP - คอนสตรักเตอร์ (Constructor)

**Constructor (คอนสตรักเตอร์)** คือฟังก์ชันพิเศษประเภทหนึ่งภายในคลาสที่ระบบของ PHP จะ**เรียกใช้งานโดยอัตโนมัติทันที**ที่มีการสร้างออบเจกต์ใหม่ (เมื่อใช้คีย์เวิร์ด `new`) 

ประโยชน์หลักของมันคือ ช่วยให้เราสามารถตั้งค่าเริ่มต้น (Initialize) ให้กับคุณสมบัติ (Properties) ของวัตถุได้ทันทีตอนที่สร้าง โดยไม่ต้องมานั่งเรียกใช้ฟังก์ชันเซตค่าทีละตัวให้เสียเวลา

---

## 🏗️ โครงสร้างคำสั่ง (Syntax)

ใน PHP เราจะประกาศ Constructor โดยใช้เครื่องหมายอันเดอร์สกอร์ 2 ตัว (`__`) ตามด้วยคำว่า `construct` เสมอ:

```php
class ClassName {
  function __construct() {
    // โค้ดที่ต้องการให้ทำงานอัตโนมัติตอนสร้าง Object
  }
}
