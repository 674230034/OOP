# PHP OOP - เมธอดสแตติก (Static Methods)

โดยปกติแล้ว เวลาเราต้องการเรียกใช้งานฟังก์ชัน (Method) ที่อยู่ภายในคลาส เราจำเป็นต้องสร้างออบเจกต์ขึ้นมาก่อนด้วยคีย์เวิร์ด `new` เสมอ 

แต่สำหรับ **Static Methods** มันคือฟังก์ชันของคลาสที่ถูกประกาศให้เป็นคุณสมบัติส่วนกลาง ทำให้เรา**สามารถเรียกใช้งานฟังก์ชันนั้นได้ทันทีโดยไม่ต้องสร้าง Object** ช่วยให้ประหยัดหน่วยความจำและเรียกใช้งานได้รวดเร็ว เหมาะกับฟังก์ชันประเภท Utility หรือฟังก์ชันคำนวณส่วนกลาง

---

## 🏗️ โครงสร้างคำสั่ง (Syntax)

เราใช้คีย์เวิร์ด **`static`** นำหน้าคำว่า `function` และใช้เครื่องหมาย **`::` (Scope Resolution Operator)** ในการเรียกใช้งาน

```php
class ClassName {
  public static function staticMethod() {
    // โค้ดที่ต้องการให้ทำงาน
  }
}

// วิธีเรียกใช้งานจากภายนอกคลาส (ไม่ต้องใช้ new)
ClassName::staticMethod();
