# PHP Namespaces (เนมสเปซ)

เมื่อโปรเจกต์ของคุณใหญ่ขึ้น ปัญหาที่พบบ่อยคือ **"ชื่อคลาสซ้ำกัน"** (เช่น คุณสร้างคลาส `User` และมี Library อื่นที่ดึงมาใช้ชื่อ `User` เหมือนกัน ทำให้ระบบ Error)

**Namespaces** ถูกสร้างขึ้นมาเพื่อแก้ปัญหานี้ โดยทำหน้าที่เหมือน **"โฟลเดอร์จำลองในโค้ด"** เพื่อจัดกลุ่มคลาสที่เกี่ยวข้องกันไว้ด้วยกัน ต่อให้ชื่อคลาสซ้ำกัน แต่อยู่คนละ Namespace ก็สามารถทำงานร่วมกันได้ไม่มีปัญหา

---

## 🚀 ตัวอย่างโค้ดใช้งานจริง (Example)

* คำสั่ง `namespace` ต้องอยู่**บรรทัดบนสุด**ของไฟล์ PHP เสมอ (ห้ามมีโค้ดอื่นอยู่ก่อนหน้ายกเว้น `<?php`)

```php
<?php
// ไฟล์ที่ 1: Html.php
namespace MyProject\Html;

class Table {
  public function draw() {
    echo "วาดตารางเวอร์ชัน HTML";
  }
}
?>
