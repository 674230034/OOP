# PHP OOP - เทรต (Traits)

ในภาษา PHP คลาสลูกจะสืบทอด (`extends`) คลาสแม่ได้**เพียงคลาสเดียวเท่านั้น** (Single Inheritance) ซึ่งบางครั้งทำให้สถาปัตยกรรมโค้ดไม่ยืดหยุ่น

**Traits** จึงถูกสร้างขึ้นมาเพื่อแก้ปัญหานี้ โดยเปิดโอกาสให้เราสร้างชุดเมธอดแยกไว้เป็นก้อน ๆ แล้วสั่งให้คลาสใดก็ได้ดึงกลุ่มเมธอดนี้ไปแปะใช้งานได้ทันทีเสมือนเป็นเมธอดของตัวเอง (ดึงมาใช้กี่ตัวก็ได้พร้อมกัน!)

---

## 🚀 ตัวอย่างโค้ดใช้งานจริง (Example)

เราใช้คีย์เวิร์ด **`trait`** ในการสร้าง และคีย์เวิร์ด **`use`** ภายในคลาสเพื่อนำมาใช้งาน

```php
<?php
trait Message1 {
  public function msg1() {
    echo "OOP มีความยืดหยุ่นมาก ";
  }
}

trait Message2 {
  public function msg2() {
    echo "เมื่อเราเข้าใจเทรต!";
  }
}

class Welcome {
  use Message1, Message2; // ดึงความสามารถจากทั้ง 2 เทรตมาใส่ในคลาสนี้พร้อมกัน
}

$obj = new Welcome();
$obj->msg1();
$obj->msg2();
// ผลลัพธ์: OOP มีความยืดหยุ่นมาก เมื่อเราเข้าใจเทรต!
?>
