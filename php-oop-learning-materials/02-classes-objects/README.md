# PHP OOP - คลาส และ ออบเจกต์ (Classes and Objects)

ในการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming: OOP) สิ่งสำคัญที่สุดสองสิ่งที่คุณต้องรู้จักคือ **Class (คลาส)** และ **Object (ออบเจกต์)** โดยคลาสจะทำหน้าที่เป็นเหมือน "พิมพ์เขียว" ส่วนออบเจกต์คือ "สิ่งของจริง" ที่ถูกสร้างขึ้นมาจากพิมพ์เขียวนั้น

---

## 🏗️ การเปรียบเทียบให้เห็นภาพ: Class vs. Object

ลองจินตนาการถึง **"แบบแปลนรถยนต์"**:
* **Class (คลาส):** คือ **พิมพ์เขียวหรือแบบแปลน** (ระบุโครงสร้าง โครงร่าง และสิ่งที่คุณสมบัติที่รถคันนี้ต้องมี แต่ตัวพิมพ์เขียวเองยังไม่ใช่วดยนต์จริง ๆ)
* **Object (ออบเจกต์):** คือ **รถยนต์คันจริง** ที่ถูกประกอบขึ้นมาจากพิมพ์เขียวนั้น (เช่น รถ Toyota สีแดง หรือ รถ Tesla สีดำ)

* **Class:** `Car` (แนวคิด/แม่พิมพ์)
* **Objects:** `$toyota`, `$tesla` (วัตถุที่จับต้องได้จริงในระบบ)

---

## 💻 โครงสร้างคำสั่ง (Syntax)

### 1. การสร้าง Class
เราใช้คีย์เวิร์ด `class` ในการสร้างคลาส ภายในคลาสจะประกอบไปด้วย **Properties** (ตัวแปรภายในคลาส) และ **Methods** (ฟังก์ชันภายในคลาส)

```php
class ClassName {
  // Properties (คุณสมบัติ/ตัวแปร)
  public $property_name;

  // Methods (พฤติกรรม/ฟังก์ชัน)
  function method_name() {
    // โค้ดที่ต้องการให้ทำงาน
  }
}
