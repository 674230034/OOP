# PHP OOP - ดีสตรักเตอร์ (Destructor)

**Destructor (ดีสตรักเตอร์)** คือฟังก์ชันพิเศษที่เป็นคู่ตรงข้ามกับ Constructor โดยระบบของ PHP จะ**เรียกใช้งานฟังก์ชันนี้โดยอัตโนมัติ** เมื่อออบเจกต์นั้นถูกทำลาย, สิ้นสุดการทำงานของสคริปต์ (Script Stops) หรือเมื่อออบเจกต์นั้นไม่ได้ถูกนำไปอ้างอิงใช้งานอีกต่อไปในหน่วยความจำ

ประโยชน์หลักของมันคือ ใช้สำหรับการ "เคลียร์พื้นที่" หรือล้างข้อมูลหลังจากเลิกใช้งานวัตถุนั้นแล้ว เช่น การปิดการเชื่อมต่อฐานข้อมูล (Database Connection) หรือการปิดไฟล์ที่เคยเปิดไว้

---

## 🏗️ โครงสร้างคำสั่ง (Syntax)

ใน PHP เราจะประกาศ Destructor โดยใช้เครื่องหมายอันเดอร์สกอร์ 2 ตัว (`__`) ตามด้วยคำว่า `destruct` เสมอ:

```php
class ClassName {
  function __destruct() {
    // โค้ดที่ต้องการให้ทำงานอัตโนมัติก่อนที่ Object จะถูกทำลาย
  }
}
