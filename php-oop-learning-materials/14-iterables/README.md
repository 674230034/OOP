# PHP Iterables (ข้อมูลประเภทวนลูปได้)

**Iterable** เป็น Data Type ตัวหนึ่งใน PHP (เริ่มมีตั้งแต่ PHP 7.1) เป็นคำกว้าง ๆ ที่ใช้เรียกข้อมูลกลุ่มใดก็ได้ที่**สามารถนำไปใช้กับคำสั่ง `foreach` เพื่อวนลูปดึงข้อมูลได้**

ค่าที่ถือว่าเป็น Iterable ใน PHP ประกอบด้วย:
1. ข้อมูลประเภท **Array** พื้นฐานทั้งหมด
2. วัตถุ (Object) ที่นำเอาอินเตอร์เฟส **`Iterator`** ไปใช้งาน

---

## 🚀 ตัวอย่างโค้ดใช้งานจริง (Example)

เรามักใช้คำว่า `iterable` เป็นข้อกำหนดในฟังก์ชัน (Type Hinting) เพื่อบอกว่า ฟังก์ชันนี้ยอมรับข้อมูลอะไรก็ได้ที่เอาไปวนลูป `foreach` ได้

```php
<?php
// ฟังก์ชันนี้บอกว่า ข้อมูลที่ส่งเข้ามาในช่อง $myList ต้องวนลูปได้นะ
function printIterable(iterable $myList) {
  foreach($myList as $item) {
    echo $item . " ";
  }
}

$arr = ["A", "B", "C"];
printIterable($arr); // ผลลัพธ์: A B C (เพราะ Array เป็น iterable โดยธรรมชาติ)
?>
