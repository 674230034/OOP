# PHP OOP - ค่าคงที่ในคลาส (Class Constants)

**Class Constants** คือการกำหนดค่าคงที่ไว้ภายในคลาส ซึ่งค่านี้จะ**ไม่มีวันเปลี่ยนแปลงได้**ตลอดการทำงานของโปรแกรม มีประโยชน์ในการเก็บข้อมูลที่เป็นค่ามาตรฐาน (เช่น ค่า PI, ข้อความแสดงสถานะ)

* เราใช้คีย์เวิร์ด **`const`** ในการประกาศ (ไม่ต้องมีเครื่องหมาย `$` นำหน้าชื่อตัวแปร)
* ค่าคงที่ของคลาสจะมีคุณสมบัติเป็น Static โดยอัตโนมัติ หมายความว่า**สามารถเรียกใช้ได้เลยโดยไม่ต้องสร้าง Object** ---

## 🚀 ตัวอย่างโค้ดใช้งานจริง (Example)

เราใช้เครื่องหมาย **`::` (Scope Resolution Operator)** ในการเข้าถึงค่าคงที่

```php
<?php
class Goodbye {
  const LEAVING_MESSAGE = "ขอบคุณที่ใช้บริการคลาสของเรา!";
  
  public function byebye() {
    // การเรียกใช้ค่าคงที่จาก "ภายใน" คลาส ให้ใช้ self::
    return self::LEAVING_MESSAGE;
  }
}

// การเรียกใช้จาก "ภายนอก" คลาส เรียกผ่านชื่อคลาสได้เลย!
echo Goodbye::LEAVING_MESSAGE; 
// ผลลัพธ์: ขอบคุณที่ใช้บริการคลาสของเรา!
?>
